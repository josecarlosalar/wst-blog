  <h1>Cómo Integrar Claude Code en VS Code en 2026: Guía Paso a Paso con Ejemplos Reales</h1>
  <p>En esta guía completa, te mostraremos cómo integrar Claude Code en Visual Studio Code en 2026. Desde la instalación básica hasta la configuración avanzada, pasando por ejemplos prácticos y soluciones a problemas comunes, este tutorial te ayudará a optimizar tu flujo de trabajo con la potencia de Claude Code.</p>
  <section id='requisitos-previos'>
    <h2>Requisitos previos: Versiones, suscripciones y configuración inicial</h2>
    <p>Antes de comenzar, asegúrate de cumplir con los siguientes requisitos:</p>
    <ul>
      <li><strong>Visual Studio Code 1.98.0 o superior</strong>: Asegúrate de tener la última versión de VS Code instalada.</li>
      <li><strong>Suscripción Claude Pro o Max</strong>: Necesitarás una suscripción activa de Claude Pro o Max, que cuesta entre $20 y $35 al mes.</li>
      <li><strong>Configuración de API keys</strong>: Accede a la consola de Anthropic para generar y configurar tu API key.</li>
    </ul>
    <p>Para generar tu API key, sigue estos pasos:</p>
    <ol>
      <li>Accede a la <a href='https://console.anthropic.com' target='_blank' rel='noopener noreferrer'>consola de Anthropic</a>.</li>
      <li>Navega a la sección de API keys.</li>
      <li>Genera una nueva API key y cópiala.</li>
    </ol>
    <p>Una vez que tengas tu API key, estarás listo para instalar la extensión de Claude Code en VS Code.</p>
  </section>
  <section id='instalacion'>
    <h2>Instalación de la extensión oficial: Del marketplace a tu primer 'Hola Mundo'</h2>
    <p>La instalación de la extensión oficial de Claude Code es sencilla y rápida. Sigue estos pasos para comenzar:</p>
    <ol>
      <li><strong>Abre Visual Studio Code</strong> y ve a la pestaña de extensiones (puedes usar el atajo <code>Ctrl+Shift+X</code>).</li>
      <li>En el buscador de extensiones, escribe <em>Claude Code</em>.</li>
      <li>Selecciona la extensión oficial (con más de 2 millones de instalaciones) y haz clic en <strong>Instalar</strong>.</li>
    </ol>
    <p>Una vez instalada la extensión, sigue estos pasos para conectar tu API key:</p>
    <ol start='4'>
      <li>Abla el panel de Claude Code en VS Code (generalmente se encuentra en la barra lateral derecha).</li>
      <li>Pega tu API key en el campo correspondiente.</li>
      <li>Haz clic en <strong>Conectar</strong>.</li>
    </ol>
    <p>Para ejecutar tu primer comando, sigue estos pasos:</p>
    <ol start='7'>
      <li>Abre un archivo de código en VS Code.</li>
      <li>Selecciona una línea de código o un bloque de código.</li>
      <li>Haz clic derecho y selecciona <strong>Explicar código</strong> o escribe <code>/explain</code> en el panel de Claude Code.</li>
    </ol>
    <p>Si encuentras el error <em>API key no válida</em>, asegúrate de que:</p>
    <ul>
      <li>La API key esté correctamente copiada y pegada.</li>
      <li>Tu suscripción Claude Pro o Max esté activa.</li>
      <li>No hayas excedido el límite de tokens permitido.</li>
    </ul>
  </section>
  <section id='configuracion-avanzada'>
    <h2>Configuración avanzada: Personalización de la barra lateral y shortcuts</h2>
    <p>Para personalizar la barra lateral y los shortcuts de Claude Code, sigue estos pasos:</p>
    <ol>
      <li><strong>Mover la extensión a la barra lateral</strong>:</li>
      <li>Ve a <strong>Configuración</strong> (puedes usar el atajo <code>Ctrl+,</code>).</li>
      <li>Busca <em>Claude Code</em>.</li>
      <li>Selecciona <em>Preferido ubicación</em> y elige <strong>Barra lateral</strong>.</li>
    </ol>
    <p>Para crear shortcuts personalizados, sigue estos pasos:</p>
    <ol start='5'>
      <li>En la misma sección de configuración, busca <em>Shortcuts</em>.</li>
      <li>Selecciona el comando que deseas personalizar (por ejemplo, <em>Explicar código</em>).</li>
      <li>Asigna un nuevo atajo de teclado (por ejemplo, <code>Ctrl+Shift+C</code>).</li>
    </ol>
    <p>Para configurar el autostart, agrega el siguiente código a tu archivo <code>settings.json</code>:</p>
    <pre><code>"claude-code.autostart": true</code></pre>
    <p>Guarda los cambios y reinicia VS Code para aplicar la configuración.</p>
  </section>
  <section id='proyectos-reales'>
    <h2>Trabajando con proyectos reales: Selección de código, diffs y commits</h2>
    <p>Claude Code puede ayudarte a trabajar con proyectos reales de varias maneras. Aquí tienes algunos ejemplos prácticos:</p>
    <h3>Seleccionar una función y pedir explicación</h3>
    <p>Selecciona una función en tu código y haz clic derecho. Elige <strong>Explicar código</strong> o escribe <code>/explain</code> en el panel de Claude Code. La IA te proporcionará una explicación detallada de la función seleccionada.</p>
    <h3>Generar tests unitarios</h3>
    <p>Selecciona una función o clase y haz clic derecho. Elige <strong>Generar tests unitarios</strong> o escribe <code>/generate-tests</code> en el panel de Claude Code. La IA generará tests unitarios para el código seleccionado.</p>
    <h3>Crear pull requests con descripciones automáticas</h3>
    <p>Cuando estés listo para hacer un pull request, selecciona los archivos modificados y haz clic derecho. Elige <strong>Crear pull request</strong> o escribe <code>/create-pr</code> en el panel de Claude Code. La IA generará una descripción automática para tu pull request.</p>
    <p>Aquí tienes un ejemplo de un diff generado por Claude Code:</p>
    <pre><code>diff --git a/src/components/Button.js b/src/components/Button.js
index 42d23f0..9e334f6 100644
--- a/src/components/Button.js
+++ b/src/components/Button.js
@@ -1,5 +1,5 @@
-const Button = ({ children, onClick }) => {
+const Button = ({ children, onClick, disabled }) => {
    return (
-        <button onClick={onClick}>{children}</button>
+        <button onClick={onClick} disabled={disabled}>{children}</button>
    );
}</code></pre>
    <h3>Integración con Git: Automatizando reviews y mensajes de commit</h3>
    <p>Claude Code puede ayudarte a automatizar reviews y mensajes de commit. Aquí tienes algunos ejemplos:</p>
    <h4>Revisar cambios antes de commit</h4>
    <p>Antes de hacer un commit, selecciona los archivos modificados y haz clic derecho. Elige <strong>Revisar cambios</strong> o escribe <code>/review-changes</code> en el panel de Claude Code. La IA revisará los cambios y te proporcionará sugerencias.</p>
    <h4>Generar mensajes de commit semánticos</h4>
    <p>Selecciona los archivos modificados y haz clic derecho. Elige <strong>Generar mensaje de commit</strong> o escribe <code>/generate-commit-message</code> en el panel de Claude Code. La IA generará un mensaje de commit semántico.</p>
    <h4>Detectar conflictos potenciales</h4>
    <p>Selecciona los archivos modificados y haz clic derecho. Elige <strong>Detectar conflictos</strong> o escribe <code>/detect-conflicts</code> en el panel de Claude Code. La IA detectará conflictos potenciales y te proporcionará sugerencias.</p>
    <p>Para configurar hooks de Git, agrega el siguiente código a tu archivo <code>.git/hooks/pre-commit</code>:</p>
    <pre><code>#!/bin/sh
echo "Revisando cambios con Claude Code..."
claude-code review-changes
if [ $? -ne 0 ]; then
echo "Error: Cambios no revisados. Por favor, corrige los errores antes de hacer commit."
  exit 1
fi</code></pre>

    <h2>Skills y agentes en VS Code: Potenciando tu flujo de trabajo</h2>
    <p>Claude Code ofrece una variedad de skills y agentes que pueden potenciar tu flujo de trabajo. Aquí tienes algunos ejemplos:</p>
    <h3>Usar skills</h3>
    <p>Para usar una skill, escribe <code>/skill</code> en el panel de Claude Code y selecciona la skill que deseas usar. Por ejemplo, puedes usar la skill <em>frontend-design</em> para obtener sugerencias de diseño frontend o la skill <em>security-review</em> para revisar la seguridad de tu código.</p>
    <h3>Configurar agentes</h3>
    <p>Para configurar un agente, escribe <code>/agent</code> en el panel de Claude Code y selecciona el agente que deseas configurar. Por ejemplo, puedes configurar un agente para refactorizar automáticamente tu código.</p>
    <p>Aquí tienes un ejemplo de un prompt para refactorizar automáticamente tu código:</p>
    <pre><code>Refactoriza la siguiente función para mejorar su legibilidad y rendimiento:

function calculateTotal(items) {
    let total = 0;
    for (let i = 0; i < items.length; i++) {
        total += items[i].price * items[i].quantity;
    }
    return total;
}</code></pre>
    <h3>Casos de uso avanzados: Debugging y arquitectura</h3>
    <p>Claude Code puede ayudarte con casos de uso avanzados como debugging y arquitectura. Aquí tienes algunos ejemplos:</p>
    <h4>Debugging de race conditions</h4>
    <p>Selecciona el código que deseas debuggear y haz clic derecho. Elige <strong>Debuggear</strong> o escribe <code>/debug</code> en el panel de Claude Code. La IA te proporcionará sugerencias para solucionar race conditions.</p>
